一. 安装依赖

  • 为了能够在Ubuntu/Debian上运行aapt(Android Asset Packaging Tool)，你需要安装ia32-libs，如果没有可以按照下面的方式安装依赖：
sudo apt-get install ia32-libs
# 如果上面的命令没有成功，可以执行：
sudo apt-get install lib32z1 lib32ncurses5 lib32bz2-1.0 lib32stdc++6
  • 需要安装Java依赖，ubuntu安装命令如下：
sudo add-apt-repository ppa:webupd8team/java
sudo apt-get update
sudo apt-get install oracle-java8-installer
  • 需要安装Java依赖，Debian安装命令如下：
#编辑一个新的ppa文件
sudo vim /etc/apt/sources.list.d/java-8-debian.list
#往文件中添加以下内容
deb http://ppa.launchpad.net/webupd8team/java/ubuntu trusty main deb-src http://ppa.launchpad.net/webupd8team/java/ubuntu trusty main
#导入GPG key
sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys EEA14886
#使用以下命令来更新apt缓存并安装java8
sudo apt-get update $ sudo apt-get install oracle-java8-installer
#使用以下命令来更新apt缓存并安装java8
sudo apt-get install oracle-java8-set-default
